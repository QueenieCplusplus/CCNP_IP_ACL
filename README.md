# CCNP IP ACL
IP Access Lists


# IP Standard Access List 標準存取列表

* 常用指令：

access-list 存取列表

acess-list-number 存取列表號碼（預設為 1-99, 延伸號碼為 100-199）

permit 允許

deny 拒絕

Source 來源IP位址

Source-wildcard 來源的運算位元

any 任何位置，代表 Source address 0.0.0.0 和 Source-wildcard 255.255.255.255

# IP Extended Access List 延伸存取列表

* 延伸目的

提供 SMTP 和 DNS 的服務
